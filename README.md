# Joshin Portfolio - Matrix React (Vite)

This is a Vite + React starter project that applies a Matrix-style theme to your portfolio.

## How to run locally

1. Install dependencies
```bash
cd joshin-portfolio-matrix-react
npm install
```

2. Run dev server
```bash
npm run dev
```

3. Build for production
```bash
npm run build
npm run preview
```

Files added/modified:
- src/App.jsx - Main React component (matrix theme)
- src/main.jsx - React entry
- src/styles.css - Matrix styling and layout
- public/assets/avatar.jpg - Avatar (copied from your upload if found)

If you want the original HTML converted into React pages as well, tell me and I will integrate them into routes.
